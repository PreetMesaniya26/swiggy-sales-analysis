# 🍔 Swiggy Sales Analysis

A data analysis project focused on extracting business insights from Swiggy order data using Python and visualization tools.

---

## 📌 Problem Statement

Swiggy aims to improve its business performance by analyzing order data to answer key business questions related to sales, customer behavior, and operational efficiency.

---

## 🎯 Key Objectives

- 📊 Analyze overall sales performance  
- 👥 Track customer engagement through ratings  
- ⏱️ Identify trends across time and locations  
- 🌱 Compare food preferences (Veg vs Non-Veg)  
- 🧠 Generate insights for business decision-making  

---

## 🔑 Key Performance Indicators (KPIs)

- 💰 Total Sales  
- 🧾 Average Order Value (AOV)  
- ⭐ Average Ratings  
- 📝 Total Ratings Count  
- 📦 Total Orders  

---

## 📊 Analysis & Charts Created

- 📅 Monthly Sales Trends  
- 📆 Daily Sales Trend  
- 📈 Weekly Sales Trends  
- 📊 Quarterly Performance Summary  
- 🌱 Total Sales by Food Type (Veg vs Non-Veg)  
- 🗺️ Total Sales by State  
- 🏙️ Top 5 Cities by Sales  

---
